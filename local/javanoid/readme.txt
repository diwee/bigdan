## 4. System requirements #############################################

Javanoid needs a Java enabled browser. If you have a JDK1.1 compatible
browser (such as MSIE4.0 or HotJava1.0), you'd better use a version of 
Javanoid designed for JDK1.1. Otherwise you'll have to use the version 
designed for JDK1.0, which should work with all Java enabled browsers.
